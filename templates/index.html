About This Project

This application uses a Random Forest Classifier to predict whether a passenger survived the Titanic disaster based on their characteristics. The model was trained on historical Titanic passenger data.

Features Used: Passenger Class, Sex, Age, Siblings/Spouses, and Ticket Fare

Target Variable: Survived (Yes/No)
